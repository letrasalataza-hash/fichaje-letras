# Fichaje Letras a la Taza

Aplicacion web de control horario para registrar entradas, salidas, pausas y ajustes manuales de empleados.

La app esta construida con React y Vite, guarda los datos en Supabase y permite exportar historiales e informes en CSV, HTML y PDF.

## Scripts

- `npm run dev`: inicia el servidor de desarrollo.
- `npm run build`: genera la version de produccion.
- `npm run preview`: previsualiza la build generada.
